## Introduction

A template to build things using [`Next.js`](https://nextjs.org/) and its [`standalone`](https://nextjs.org/docs/app/api-reference/config/next-config-js/output) mode.

Taking into consideration what is mentioned in the [docs](https://nextjs.org/docs/app/api-reference/config/next-config-js/output), it is defined the `standalone` mode in [`next.config.ts`](./next.config.ts).

On top of that, it is customized the [`build` script](./package.json), so it generates the respective code but also the static content.

Finally, it is customized the [`start` script](./package.json), so it uses the server generated by the `standalone` mode.

## Usage

Update the [`README.md`](README.md), specifically the badges.

Update the following [`package.json`](package.json) settings:

- `description`
- `repository.url`
- `author`
- `bugs.url`
- `homepage`
- `keywords`

Finally, to deploy your app, run:

- `npm run build`
- `npm run start`
